# Telecom Customer Churn Analysis

## Objective
Analyze telecom customer data to understand churn patterns and identify key drivers of customer attrition.

## Tools Used
Python (Pandas)
Tableau
GitHub

## Dataset
7043 telecom customers with contract, service usage, and billing information.

## Analysis Steps
1. Data cleaning using Python
2. Feature engineering (Tenure Groups, Revenue Segments)
3. Churn analysis
4. Interactive dashboard creation in Tableau

## Key Insights
• Customers with month-to-month contracts show the highest churn (~42%)
• Churn is highest among customers in their first 6 months
• Service dissatisfaction is a major churn driver
• High revenue customers show slightly higher churn risk
